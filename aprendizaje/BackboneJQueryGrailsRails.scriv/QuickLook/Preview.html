    Rails Primeros pasos

    Crear aplicación, scaffolding

    Comandos para crear aplicación, crear controlador, ejecutar migración
    rails new things_i_bought
    cd things_i_bought/
    bin/rails generate scaffold purchase name:string cost:float

    La aplicación se ejecuta mediante

    bin/rails server

    Migraciones
    bin/rake db:migrate
    bin/rake db:rollback
